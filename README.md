# Currency Exchange Rate Tracker

A Simple Desktop GUI Application written in Python which uses Real-Time-API's to Convert and Visualise global Currency Trends and Exchange Rates.

## Technologies Used:
- **Language:** Python.
- **Interface:**Tkinter.
- **Libraries**Matplotlib.

## Project Structure:
-`gui.py`: User Interface
-`converter.py`: Currency Conversion Logic
-`historical_trends.py`: Visualisation

## How to Run:
1. Clone the Repository
   git clone https://github.com/synkline/Currency-Exchange-Rate-Tracker/edit/main/README.md

2. Navigate to the Folder
   cd "Python Projekt"

3. Run the Application
   python gui.py
